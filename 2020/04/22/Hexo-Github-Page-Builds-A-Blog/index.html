Hexo + Github Page 搭建博客

发表于 2020-04-22 更新于 2020-07-14 分类于 工具 Valine：

前言

Ander’s Blog 用来记录生活，分享技巧，整理知识，构建我的知识体系。去年也曾搭建过 Blog，后来没能坚持一直更新。这次我狠下心，咬了咬牙到 GoDaddy 购买了一个域名。作为一名没有其他兼职的全职大学生，钱还是很重要的。这次一定要坚持下去，要不然对不起自己的钱。

由于武汉疫情的爆发，我已经在家里蹲了近4个月了。在没有口罩的情况下，成功苟到现在。学校通知在5月13号会开学，关于这点我还是很开心。从前的我，上网和上课是分开的，现在的我，每天在家上网课。对我来说，网课的内容完全不进脑子，我吸收不到知识。（我的作业是怎么完成的呢？）!

搭建环境

安装

我们先到 Github 创建一个账户，然后再创建一个仓库 (Repository) 。创建的仓库名必须为 yourname.github.io ，yourname 就是 GitHub 的账户名称。然后点击 Create Repository 完成创建。

接着打开电脑的终端 (Terminal) 创建一个空文件夹用来存放 Blog 的所有信息。安装 Node.js 和 Git 到电脑，安装方法很简单，到网络可以查到很多相关资料。我这里就只介绍 Mac 的安装方法。

每次修改好配置不要急着部署，先用服务器检查一下生成的页面，确认无误后再部署到远程库。

需要注意，在部署过程中需要提供远程库所在的 GitHub 账户和密码。可以使用 SSH (通过在网络中创建安全隧道来实现SSH客户端与服务器之间的连接) ，这里又可以水一篇 Blog 了。

如果在电脑中已经保存有一个 Git 账号信息，想要使用其他账号提交文件到远程库。Git 的配置信息分为三种:

  1. 系统配置: 包含了适用于系统所有用户和所有库的值
  2. 全局配置: 所有用户共同的配置
  3. 目录配置: 单一库的用户配置
