Hexo NexT 主题配置

发表于 2020-04-28 分类于 工具 Valine：

前言

Ander’s Blog 用来记录生活，分享技巧，整理知识，构建我的知识体系。 NexT 是 Hexo 中一个比较完善的 Hexo 主题，内置了很多插件供我们使用，避免了自己安装插件过程中导致的很多问题。它的配置方式也非常简单，只需要改动主题配置文件中相应的设置就可以实现各种不同的功能。官方 Blog 的更新似乎跟不上主题版本的更新，所以才有了这篇 Blog。

你们可能不知道用全部学习的时间来玩游戏是什么概念。我们一般用两个字来形容这种人: 土块。我经常说一句话: 当年 卢老爷他可以把全部学习的时间用来打游戏进入世界赛，我用学习的时间全部来打游戏拿到国标不是问题。话说最近的网课基本没听，我这学期考试可能凶多吉少了。

  2. 选择你所需要的版本，下载 Download 区域下的 Source Code (zip) 到本地。

主题配置

NexT 主题有丰富的功能，很多我也没有使用过，这里的配置是我使用过的一些，要想使用 NexT 的其他功能，可以自己找教程或者到官网查找，官网地址在文章最后。

基本配置

首先就是到站点文件中更改主题，默认主题是 landspace 。有一说一 landspace 是真的丑，也或许是一种我欣赏不了的美。直接将 landscape 改为 next 即可。

接着就是打开近 1000 行的主题配置文件了。文件的大部分是英文注释，介绍了如何开启某些选项。我们需要更改的不算多。相对于我不喜欢有太多的功能。
